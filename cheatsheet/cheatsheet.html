CSS Cheatsheet

Introduction

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

Selectors and Visual Rules

Property Description
Selectors Type Selector: p
Class Selector: .column
ID Selector: #first-item
Specificity The most specific selector type is the ID selector, followed by class selectors, followed by type selectors.
Chaining Selectors CSS selectors can be chained so that rule sets apply only to elements that match all criteria. Select h3 elements with the section-heading class: h3.section-heading
Font Size font-size: 30px;
Background Color background-color: blue;
Opacity The value of this property ranges from 0 (transparent) to 1 (opaque). opacity: 0.5;
Font Weight The provided value can be a keyword such as bold or normal. font-weight: bold;
Text Align This property can be set to these values: left, right, or center. text-align: right;
